# 3차 발표
### 게임에 대한 간단한 소개
플래피 버드와 비슷한 형식의 게임 <br>
위 게임과 달리 체력이 0이 되면 게임이 종료되고, 여러 캐릭터가 있으며 각 캐릭터 별 능력치와 스킬이 존재함.

### 사용된 기술
+ 개발 도구 : Kotlin, AndroidStudio

+ 참고한 것
  1. 수업 시간에 배운 내용(ImageSwitcher 등)
  2. ChatGPT 및 인터넷 자료

+ 수업 내용에서 차용한 것
  1. 일부 리소스
  2. a2dg

+ 직접 개발한 것
 + 캐릭터별 능력치 및 스킬 설계
 + 장애물 생성 및 난이도 증가 시스템
 + 플레이어-장애물 충돌 판정 처리

### 아쉬운 것들
+ 하고 싶었지만 못한 것
: 캐릭터 선택 화면에 능력치 및 스킬 표기

+ 결국 해결하지 못한 문제/버그
: 게임 시작 후 첫 번째 화면 터치 시에 캐릭터가 올라가면서 나오는 소리가 나오지 않음.

+ 기말 프로젝트를 하면서 겪은 어려움
: 캐릭터 별 밸런스, 티가 나지 않는 난이도 상승과 캐릭터 능력치
