Project 소개
---------
+ Project 제목: Fly!
+ 게임 컨셉
  + 캐릭터의 앞에서 다가오는 기둥을 피하며 살아남아야 함.
  + 화면 우측에서부터 좌측으로 기둥이 움직임.
  + 기둥은 상단과 하단에 위치함.
  + 상단 기둥과 하단 기둥 사이 캐릭터가 지나갈 수 있을 만한 공백이 있음.
  + 화면을 터치하면 캐릭터가 일정 높이만큼 올라감.
  + 화면을 터치하지 않으면 터치 입력이 있을 때까지 캐릭터는 점점 내려감.
  + 캐릭터의 체력이 0이 되면 게임 아웃
  + 오래 생존할수록 난이도 상승 (기둥이 다가오는 속도, 상하 기둥 사이 공백 조절)
  + 지나간 기둥 수 만큼 점수 올라감.
